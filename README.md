# Airbnb Landing Page (HTML & CSS Study)

## 📌 About the Project

This project is a front-end study that recreates the main interface of the Airbnb landing page using only HTML and CSS. The goal was to practice building a structured and visually organized web page while focusing on layout, styling, and component organization.

The project reproduces several elements commonly found in modern web interfaces such as navigation menus, search sections, content cards, and a footer layout.

## 🎯 Objective

The main objective of this project was to strengthen the fundamentals of front-end development by working with:

- Semantic HTML structure
- Layout organization
- CSS styling
- Component-based page sections

## 🛠 Technologies Used

- HTML5
- CSS3

## 🧠 Concepts Practiced

During the development of this project, the following concepts were practiced:

- Semantic page structure
- Layout construction using **Flexbox**
- Interface component organization
- Typography customization using **@font-face**
- Responsive visual structuring concepts
- UI composition inspired by a real-world platform
